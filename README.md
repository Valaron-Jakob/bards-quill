# Bard's Quill

This simple web app is a small tool for Minecraft roleplay enthusiasts, to simplify the writing of longer texts a bit.

The tool is created with Googles AI Studio as well as GitHub Copilot and contains just a hand full of manual changes to fit everything in place.

> There is no license, so do whatever you want with it.

## Run locally

**Prerequisites:**  Node.js

1. Clone the repo via GitHub
2. Install dependencies: `npm install`
3. Run the app: `npm run dev`

## Build

1. Run the build script: `npm run build`
2. Files appear inside the `dist` folder (there are just two :D)
3. Start the preview: `npm run preview`
